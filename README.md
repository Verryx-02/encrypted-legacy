# encrypted-legacy

Questa repo ospita un file cifrato contenente dati personali riservati.

Cifrato con [age](https://github.com/FiloSottile/age) v1.3.1.  
L'unico modo per decifrarlo è mettere insieme almeno 4 dei 7 frammenti della pasphrase usando lo [Shamir's Secret Sharing Scheme v0.5](http://point-at-infinity.org/ssss/).

Se sei qui, probabilmente sai cosa fare.   
Se non sai cosa fare, probabilmente non dovresti essere qui.
